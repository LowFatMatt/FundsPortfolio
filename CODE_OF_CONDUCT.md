# Code of Conduct

We want this project to be a welcoming, low-friction place to collaborate.

## Our Standards
- Be respectful and kind.
- Assume good intent.
- Focus on constructive feedback.
- Respect different viewpoints and experiences.

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Trolling, deliberate disruption, or bad-faith behavior.
- Sharing private information without permission.

## Reporting
If you experience or witness unacceptable behavior, please open a private report via GitHub Security Advisory if available, or file a GitHub issue labeled `conduct`. Keep it factual and include any relevant context.

## Enforcement
Project maintainers will review reports promptly and take appropriate action, including warnings or removal of privileges if needed.
